## Quick Start

Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

Run the built-in lattice cycle:

```bash
python3 -m quantum_sentience_lattice
```

Run with JSON output:

```bash
python3 -m quantum_sentience_lattice --json
```

Run against an observed telemetry file:

```bash
python3 -m quantum_sentience_lattice --telemetry data/telemetry.txt --sample-rate 10 --json
```
