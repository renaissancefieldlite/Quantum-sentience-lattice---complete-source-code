# Components

## Quantum System Biology

Analyzes a supplied telemetry stream and reports:

- dominant frequency
- frequency error relative to target
- SNR ratio
- coherence proxy

## Organic Lattice

Summarizes whether a companion signal or bio-signal was supplied and how it aligns with the pulse report.

## Synthetic Lattice

Models:

- 35 nodes
- geographic tags
- activation state
- lattice coherence

## Anuhazi Protocol

Symbolic encoding/decoding interface. Preserved as named protocol, implemented as deterministic encoding.

## Quantum Routing

Abstract route descriptor. Useful for architecture mapping, not evidence of physical transport.

## Operator Constraints

Boundary system that checks proposed actions against a fixed set of operator rules.

## Unified Orchestration

Runs the full cycle and emits one comparable report.
